CASA INTELIGENTE

Ofrecemos kits educativos diseñados para colegios, donde los estudiantes aprenderán conceptos básicos de electrónica y programación al construir y personalizar un modelo funcional de una casa inteligente.

La Casa Mágica de Carlitos

El proyecto "La Casa Mágica de Carlitos" está diseñado para enseñar electrónica y programación a estudiantes a partir de 9 años. A través de un kit educativo basado en el microcontrolador ESP32, los estudiantes realizarán retos prácticos de domótica, integrando dispositivos como LEDs, pantallas LCD, sensores, ventiladores y más. Los estudiantes aprenderán a programar y conectar estos componentes para crear su propia "casa inteligente".

Este curso se estructura de manera progresiva, desde conceptos básicos de electrónica hasta el control remoto de dispositivos y la automatización de ambientes. Los niños realizarán tareas como abrir puertas con servomotores usando tarjetas RFID o controlar el encendido de ventiladores con sensores de temperatura. El curso fomenta la personalización y creatividad, permitiendo que cada estudiante diseñe su propia "casa".

El proyecto emplea una metodología activa, con clases de una hora semanal durante seis meses. Al finalizar, los estudiantes no solo habrán aprendido electrónica, sino también programación y habilidades de resolución de problemas, preparándolos para tecnologías más avanzadas.

Descubre el Kit Educativo "Casa Inteligente", una herramienta interactiva que permite a los estudiantes aprender electrónica, programación y conceptos de domótica al construir y personalizar su propio modelo funcional de una casa inteligente. Diseñado para inspirar creatividad y pasión por la tecnología, este kit transforma el aprendizaje en una experiencia práctica y divertida.

¿Qué incluye el kit?

Nuestro kit contiene todo lo necesario para construir una casa inteligente funcional:

  • ESP32 con pantalla integrada: Para programar y controlar la casa inteligente.
  • Módulo RFID con tarjeta y lector: Ideal para aprender sobre acceso y sistemas de autenticación.
  • Luces LED: Para implementar sistemas de iluminación automatizados.
  • Botones: Perfectos para figuras humanas interactivas.
  • Sensor de temperatura y humedad: Para monitorizar y controlar el ambiente de la casa.
  • Motorreductor: Para simular movimiento automatizado, como puertas o ventanas.
  • Ventilador: Para proyectos de ventilación mecánica.
  • Buzzer: Para alarmas o notificaciones interesantes.

¡Todos pensados para garantizar una experiencia educativa completa!

Beneficios Educativos

Este kit permite que los estudiantes:

  • Aprenden conceptos básicos de programación y electrónica.
  • Desarrollan habilidades de resolución de problemas y pensamiento crítico.
  • Construyan y personalicen un modelo funcional de casa inteligente.
  • Se familiaricen con tecnologías modernas como RFID y microcontroladores.

Funcionalidades de la casa inteligente

Con el kit, los estudiantes podrán:

  • Diseñar sistemas de acceso seguro usando el módulo RFID.
  • Programar iluminación automática con LEDs.
  • Controlar la temperatura y ventilación usando sensores y ventiladores.
  • Crear alarmas con el buzzer y botones interactivos.
  • Integrar sistemas de movimiento automatizado con el motorreductor.

Esto permitirá combinar creatividad, lógica y tecnología de manera práctica.
