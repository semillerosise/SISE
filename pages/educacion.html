Educación

¿Qué son los Sistemas Embebidos?
Los sistemas embebidos son dispositivos electrónicos diseñados para realizar funciones específicas dentro de un sistema mayor. Combinan hardware y software optimizados para tareas como el control, monitoreo o automatización en aplicaciones como electrodomésticos, automóviles, dispositivos médicos y sistemas industriales.
¿Qué es Arduino y para qué sirve?
Arduino es una compañía de desarrollo de software y hardware libres, así como una comunidad internacional que diseña y manufactura placas de desarrollo de hardware para construir dispositivos digitales y dispositivos interactivos que puedan detectar y controlar objetos del mundo real.

Kits Educativos

Kit Educativo Casa Domótica

El proyecto "La Casa Mágica de Carlitos" es un kit educativo de casa domótica con ESP32, diseñado para enseñar electrónica y programación a niños a partir de 9 años. Utiliza la metodología STEAM para combinar ciencia, tecnología, ingeniería, arte y matemáticas, ofreciendo una experiencia interactiva y didáctica mediante retos prácticos.

Kit Educativo Invernadero

El Invernadero Inteligente es un proyecto educativo STEAM que enseña a niños y jóvenes sobre el Internet de las Cosas (IoT) mediante aplicaciones reales como los cultivos verticales, fomentando la innovación, el pensamiento crítico, la sostenibilidad y la conciencia ambiental, mientras se exploran soluciones tecnológicas prácticas.
